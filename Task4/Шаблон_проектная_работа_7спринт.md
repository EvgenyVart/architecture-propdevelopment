| Роль  | Права роли | Группы пользователей |
| --- | --- | --- |
| cluster-viewer      | `get`, `list`, `watch` для всех ресурсов (кроме Secrets)                   | Аналитики, Менеджеры               |
| namespace-editor    | `get`, `list`, `watch`, `create`, `update`, `delete` в пределах namespace  | Разработчики, Тестировщики         |
| security-auditor    | `get`, `list` для Secrets и ConfigMaps                                     | Отдел ИБ                           |
| ci-operator         | `create`, `get`, `list`, `delete` для Pods/Jobs в CI-неймспейсах           | CI/CD системы                      |
| cluster-admin       | Полные права (`*`) на все ресурсы                                          | Администраторы                     |
| log-reader          | `get`, `list` для логов Pods и событий                                     | Служба поддержки                   |
| network-admin       | Управление NetworkPolicies и Ingress (`create`, `patch`, `delete`)         | Сетевые инженеры                   |
